# Work Order Log Form (Baseline Period)

> **Purpose**: Track work order creation, completion times, and duplicates during the 2-week baseline period. This data will establish how long work orders take WITHOUT QuarryCMMS, and identify duplicate/lost work orders.

---

## Instructions

1. **Log EVERY work order** created during baseline period
2. **Record completion time** when work is finished
3. **Mark duplicates** if the same work was requested multiple times
4. **Submit to supervisor** at end of each week

## How to Calculate Duration

**Formula**: Completed Time - Created Time = Duration

**Example**:
- Created: Jan 15, 8:00 AM
- Completed: Jan 15, 2:30 PM
- Duration: 6.5 hours

**Tips**:
- If work spans multiple days, include all time (e.g., 24+ hours)
- Round to nearest 0.5 hour
- Include time waiting for parts

---

## What is a Duplicate Work Order?

Mark "Y" for duplicate if:

1. **Same equipment, same problem** was reported by another person
2. **Work was already in progress** when WO was created
3. **Someone else already fixed it** before you got there

**Why track duplicates?**
- Duplicates waste time and create confusion
- QuarryCMMS will show real-time status to prevent duplicates
- We need to know how many duplicates happen WITHOUT the app

## What Goes in Notes Column

Brief notes to help supervisor understand:

| Situation | Example Note |
|-----------|--------------|
| Still waiting | "Parts on order" |
| Emergency | "Emergency - line down" |
| Handed off | "Passed to night shift" |
| Cancelled | "Not needed - false alarm" |
| Quality issue | "Had to redo work" |

---

## Open Work Orders at Week End

If a work order is **not completed** by end of week:

1. Leave "Completed" and "Duration" columns blank
2. Write status in Notes (e.g., "In progress", "Waiting parts")
3. Carry over to next week's form
4. Mark completion when finished

---

## Don't Have WO Numbers?

If your site doesn't currently use WO numbers:

1. Create your own: `BL-001`, `BL-002`, etc. (BL = Baseline)
2. Keep sequential within the week
3. Write brief description in Notes
