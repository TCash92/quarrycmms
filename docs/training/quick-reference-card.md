## Print Specifications

| Property | Value |
|----------|-------|
| Paper Size | Letter (8.5" x 11") or A4 |
| Orientation | Landscape |
| Margins | 0.5" all sides |
| Font | Sans-serif (Arial, Helvetica) |
| Minimum Font Size | 10pt for body, 12pt for headers |
| Lamination | 5 mil thickness recommended |
| Color | Full color recommended; B&W acceptable |

## Distribution

- Provide one card per field technician
- Post enlarged version (11" x 17") in break room
- Include in new hire orientation packet
- Replace annually or when app updates significantly
